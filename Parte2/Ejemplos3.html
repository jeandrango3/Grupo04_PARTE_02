Hay dos formas de propagación de eventos en HTML DOM, burbujeo y captura.La propagación de eventos es una forma de definir el orden de los elementos cuando ocurre un evento. Si tiene un elemento "p" dentro de un elemento "div" y el usuario hace clic en el elemento "p", ¿qué evento de "clic" de elemento debe manejarse primero?. Al burbujear , el evento del elemento más interno se maneja primero y luego el externo: el evento de clic del elemento "p" se maneja primero, luego el evento de clic del elemento "div". Al capturar el evento del elemento más externo se maneja primero y luego el interno: el evento de clic del elemento "div" se manejará primero, luego el evento de clic del elemento
